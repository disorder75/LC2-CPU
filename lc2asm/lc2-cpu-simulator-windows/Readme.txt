LC-2 Software README.TXT
November 12, 2000

Contents
--------
The LC-2 Software for Windows Distribution
contains the following files:

Simulate.exe - the LC-2 Simulator
Simulate.hlp - Simulator help file
Simulate.cnt - Simulator help file
LC2Edit.exe  - the LC-2 Editor
LC2Edit.hlp  - LC2Edit help file
LC2Edit.cnt  - LC2Edit help file
Readme.txt   - this file


Installation and Usage
----------------------
The LC-2 Software does not require a specific installation.  You may run
it directly off of the CD or copy it to your hard drive for easier access.

Directions and help information are available for both the simulator and the
editor by selecting the "Contents" option from the "Help" menu in each
respective program.
